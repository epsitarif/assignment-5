DOM TASK

  1. What is DOM and what is the purpose of DOM?
    2. ans: DOM হচ্ছে ডকুমেন্ট অবজেক্ট মডেল (Document Object Model) যা Html কে সহজেই রিপ্রেজেন্ট করে এবং Porgramming Language দিয়ে সহজেই মডিফাই করা যায়

  2. How will you select HTML elements using DOM? Name the DOM methods.

    3. Ans: এই ৫ টি উপায়ে আমরা ডম এর সাহায্যে HTML Elements ধরতে পারি
    1. getElementsByTagName()
    2. getElementsByClassName()
    3. getElementById()
    4. querySelector()
    5. querySelectorAll()


  3. What is event bubble?
    4. Ans: ইভেন্ট বাবলিং হলো ইভেন্ট-এর প্রোপাগেশন যেটা এর অরিজিন থেকে উপরে রুট ইলিমেন্ট পর্যন্ত গিয়ে থামে।
  4. What is a callback function and why will you use it?

    5. Ans: যে Function অন্য কোন Function Execute হওয়ার পরে Execute হয় তাকে CallBack Function বলে। CallBack Function কোড কে সময় মতো রান করে তাই এরর পাওয়ার সম্ভাবনা থাকে নাহ।
